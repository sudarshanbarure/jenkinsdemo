# jenkinsdemo
18/11/2025

Task1:
 1. Create a new Git repository.
 2. Create two branches: branch1 and branch2.
 3. In branch1, add an app.py that prints "This is from branch1".
 4. In branch2, add an app.py that prints "This is from branch2".
 5. Create a Jenkins pipeline with a branch selection parameter.
 6. Pipeline should clone the selected branch.
 7. Pipeline should run app.py and show the output in Jenkins console.



 Task 2:
 1. Create a new user (eg developer) in Jenkins.
 2. Give the user permission to view Jenkins jobs.
 3. Allow the user to trigger and build jobs.
 4. Ensure the user cannot modify or change any job configuration.
 5. Verify the user has read-only access with build privileges only.
